# 🚀 qwq AIOps 平台 - 快速开始

## 一键部署（推荐）⭐

只需要一个命令：

```bash
chmod +x 一键部署.sh
sudo ./一键部署.sh
```

**就这么简单！** 脚本会自动完成所有配置和部署。

## 预期时间

- **首次部署**：6-10 分钟
- **后续部署**：1-2 分钟

## 部署成功后

访问以下地址：

| 服务 | 地址 | 说明 |
|------|------|------|
| **前端界面** | http://localhost:8081 | 主界面 |
| **API 文档** | http://localhost:8081/api/docs | Swagger UI |
| **Prometheus** | http://localhost:9090 | 监控指标 |
| **Grafana** | http://localhost:3000 | 可视化面板 |

**默认账号**：
- 用户名：`admin`
- 密码：`admin123`

## 脚本做了什么

`一键部署.sh` 会自动：

1. ✅ 配置 Docker 国内镜像源（加速下载）
2. ✅ 创建所需的配置文件
3. ✅ 构建 Docker 镜像
4. ✅ 启动所有服务
5. ✅ 验证服务状态

## 常用命令

```bash
# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f qwq

# 停止服务
docker-compose down

# 重启服务
docker-compose restart

# 重新部署
sudo ./一键部署.sh
```

## 如果遇到问题

### 问题 1：网络超时

```bash
# 检查网络
ping docker.io

# 手动拉取镜像
docker pull node:18-alpine
docker pull golang:1.23-alpine
docker pull alpine:3.19

# 重新运行
sudo ./一键部署.sh
```

### 问题 2：端口被占用

编辑 `docker-compose.yml`，修改端口：

```yaml
ports:
  - "8082:8080"  # 改为其他端口
```

### 问题 3：磁盘空间不足

```bash
# 检查空间
df -h

# 清理 Docker
docker system prune -a
```
